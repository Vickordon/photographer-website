# 🚀 Швидкий старт

## Одна команда для запуску

```bash
chmod +x start.sh
./start.sh
```

Це все! Додаток:
1. ✅ Збере Docker образи
2. ✅ Створить базу даних
3. ✅ Запустить міграції
4. ✅ Заповнить тестовими даними
5. ✅ Запустить додаток

### Доступ

- **Сайт:** http://localhost:4000
- **Адмінка:** http://localhost:4000/login

### Стандартні дані для входу

```
Email:    admin@photographer.com
Пароль:   admin123
```

⚠️ **Змініть пароль після першого входу!**

## 📋 Команди

```bash
# Запуск
./start.sh

# Зупинка
./stop.sh

# Перегляд логів
docker-compose logs -f

# Зупинити і видалити дані
docker-compose down -v
```

---

## 🔧 Налаштування

### Змінні оточення

```bash
cp .env.example .env
```

Відредагуйте `.env` зі своїми налаштуваннями.

### Генерація SECRET_KEY_BASE

Автоматично генерується при першому запуску, або вручну:

```bash
mix phx.gen.secret
```

---

## 🐛 Вирішення проблем

### Порт 4000 вже використовується
```bash
lsof -ti:4000 | xargs kill -9
```

### Помилка підключення до БД
```bash
docker-compose restart db
docker-compose logs db
```

### Додаток не запускається
```bash
docker-compose build --no-cache
docker-compose logs app
```
